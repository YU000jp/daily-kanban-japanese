# かんばんボード (サイドバータイプ)

- 「かんばんボード」のアプリです (パソコン用ソフトウェア)
- インストールすると、サイドバーのように縦長のウィンドウが表示されます。
- 各曜日に、項目をマウスでドラッグして使います。

## 仕様

- OSがダークモードに設定されている場合は、ダークモードが適用されます。
- サイドバーとして、ほかのウィンドウの隣に、細く縦長に表示させる想定で作りました。
